# ChocAn Simulator

## Requirements

- Python 3.10 or later
- `mypy` (only for typechecking)

## Instructions

- Run the program:

  ```
  python -m choc_an
  ```

- Run the tests:

  ```
  python test_choc_an.py
  ```

- Type check the source code (requires `mypy`):

  ```
  mypy choc_an test_choc_an.py
  ```
